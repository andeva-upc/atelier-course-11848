# ☕ Java Fundamentals

> **Bienvenido al repositorio oficial del curso Java Fundamentals.**
> Este curso está diseñado para estudiantes de 12 a 17 años que quieren aprender a programar desde cero usando Java.

---

## 📋 Descripción del Curso

**Java Fundamentals** es un curso introductorio de programación orientado a jóvenes estudiantes. A lo largo de **6 lecciones de 10 minutos cada una**, aprenderás los conceptos esenciales de Java: desde imprimir tu primer mensaje en pantalla hasta crear objetos con atributos y métodos.

**No necesitas instalar nada.** Todo el código se ejecuta directamente en el navegador usando [Replit](https://replit.com) u [OnlineGDB](https://www.onlinegdb.com/online_java_compiler).

---

## 🎯 Objetivos del Curso

Al finalizar el curso, el estudiante será capaz de:

- ✅ Entender qué es la programación y para qué sirve Java.
- ✅ Declarar variables y trabajar con tipos de datos básicos.
- ✅ Usar estructuras de control: `if`, `for` y `while`.
- ✅ Crear y utilizar métodos reutilizables.
- ✅ Leer datos ingresados por el usuario con `Scanner`.
- ✅ Comprender los fundamentos de la Programación Orientada a Objetos (POO).
- ✅ Identificar y corregir errores comunes en código Java.

## 🚀 ¿Cómo empezar? (Configuración)

No necesitas instalar nada en tu computadora. Sigue estos pasos:

### Opción A — Usar Replit (Recomendado)
1. Ve a [https://replit.com](https://replit.com) en tu navegador.
2. Crea una cuenta gratuita (puedes usar tu cuenta de Google).
3. Haz clic en **"+ Create Repl"**.
4. En la barra de búsqueda de lenguajes, escribe **"Java"** y selecciónalo.
5. Ponle un nombre a tu proyecto (ejemplo: `MiCursoJava`).
6. Haz clic en **"Create Repl"**.
7. ¡Listo! Verás un editor con el archivo `Main.java` abierto.

### Opción B — Usar OnlineGDB
1. Ve a [https://www.onlinegdb.com/online_java_compiler](https://www.onlinegdb.com/online_java_compiler).
2. No necesitas crear cuenta para ejecutar código.
3. Escribe tu código en el área de texto central.
4. Haz clic en el botón verde **"Run"** para ejecutarlo.
5. Verás el resultado en la parte inferior de la pantalla.
